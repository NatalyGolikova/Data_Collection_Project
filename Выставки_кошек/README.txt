ДАННЫЕ С САЙТА ВЫСТАВКИ КОШЕК


Цель сбора данных - собрать информацию со всех страниц о происходивших выставках кошек 
с сайта RU-pets.ru (http://ru-pets.ru/index.php?m=6&c=2&to=1).


Данные для сбора:
• Дата проведения
• Название выставки
• Клуб-организатор


Используемый фреймворк - BeautifulSoup


Результат - csv-файл.
